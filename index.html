IT Support Knowledge Base

A simple support webpage with clear steps for common user issues, created as a beginner cloud deployment project.

Password Reset

  1. Confirm the user account email address.
  2. Ask the user to check if MFA is working.
  3. Send the password reset link through the approved system.
  4. Document the case and confirm the user can log in.

Login Issue Checklist

  • Check username and password spelling.
  • Confirm internet connection and browser access.
  • Ask the user to try a private/incognito window.
  • Check if the account is locked or disabled.

Billing or Account Questions

Collect the customer name, account ID, invoice number, and a short description of the issue. If billing access is limited, escalate the case to the correct team.

When to Escalate

Escalate if there is a service outage, repeated login failure, missing access rights, suspected security issue, or a problem that requires administrator-level access.
